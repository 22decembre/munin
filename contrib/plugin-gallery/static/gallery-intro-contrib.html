Welcome to the Munin Plugin Gallery

This is the place where you can browse description and graph images for our Munin Plugins. It is not ready and complete yet. Example graph images are still missing and many plugins lack perldoc sections. Help from contributors is welcome :-)

The gallery has two showrooms. One called Core Collection for the plugins that we deliver with the distribution of Munin-Node and one called 3rd-Party Collection for the plugins from the wild, that were uploaded to our Contrib-Repository.

Initiative to increase usability in 3rd-Party Collection

Our build script for this gallery collection reports on 2017-02-22:

	80 plugins without category were assigned to category 'other'
	321 times created perldoc pages with content
	766 times no perldoc content found
	39 example graph images illustrate the plugin pages

It is great (!) to have so many plugins, graph images and pages already, but it will need a lot of work to fill missing pages and upload images for useful plugins, that are not well presented in the gallery at the time. We will be happy if you help by adding info in perldoc format and representative example graph images to the contrib repo. The more descriptive content is there, the more helpful the Plugin Gallery will be ~

Have a look at our instructions in the Munin Wiki

To get a clear and user-friendly overview in the plugin gallery (and on Munin WebGUI), we just started with reducing the number of categories. Have a look at our list of well-known-categories, which is a proposal and can be discussed on our user mailing list.

Generic Categories

  • auth (All measurements around authentication servers and services (freeradius, kerberos, ..)
  • appserver (Application servers like Glassfish, JBoss, ..)
  • backup (All measurements around creation of backups)
  • db (Database: including MySQL, Oracle, ..)
  • cloud
  • cpu (CPU measurements)
  • disk (Disk measurements: disk space, free inodes, throughput, ..)
  • dns (bind, pdns, ..)
  • fs (Filesystem: including ZFS, samba, ..)
  • fw (Firewall: moblock, ..)
  • htc (High-throughput computing: including boinc, condor, bitcoin, ..)
  • loadbalancer (haproxy, ipvs, ..)
  • mail (Mail throughput, mail queues, etc.)
  • memory (All kind of memory measurements. Note that info about memory caching servers is also placed here)
  • munin (Monitoring the monitoring.. (includes other monitoring servers also) [shall we choose another name then? -> monitoring?]
  • network (Network efficiency, bandwidth, amounts of data transported etc.)
  • processes (Process and kernel related measurements.
  • san (Storage Area Network)
  • search (All kinds of measurement around search engines.)
  • security (Info about pending update packages for OS, ..)
  • sensors (Hardware measurements: temperature, power, devices health state, ..)
  • system
  • spamfilter (Watch your spam fighters at work. spamassassin, spamd, ..)
  • webserver (All kinds of webserver measurements for Apache, nginx and other webserver and also utilities like varnish.)

And now start browsing the plugin collection with a click on a generic category (see above) or any from the left side menu.
